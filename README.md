# 推奨環境

- node LTS 8.11.2以上
- npm 5.2.0 以上
- VS CODE
- Prettier
- CSScomb autosave
- EditorConfig

---


##  開発サーバー立ち上げ

```
npx gulp

or

npm run gulp

or

yarn gulp
```

---

## 本番ビルド

```
npm run build

or

yarn build
```

---

## gulp コマンド一覧

```
npx gulp --tasks

├── default
├── pug
├── sass

~~~
